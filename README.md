# ShipmentX v6.1 Demo

Client-demo-ready Next.js delivery of the supplied ShipmentX v6.1 operations-console experience.

## Run locally

```bash
pnpm install
pnpm dev
```

Open `http://localhost:3000`.

## Vercel

Import this repository in Vercel using the default Next.js preset. No environment variables, database, API credentials, or additional build settings are required. The normal `pnpm build` command is the production build.

## Presentation behavior

- All v6.1 navigation destinations, guided-story steps, mobile Driver/Yard Operator previews, and mockup interactions are available from the main console.
- Demo container changes persist in the browser for the duration of a presentation and after refresh.
- Use **Reset demo** in the lower-right corner to restore the opening client scenario before the next presentation.
